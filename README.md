---
**LANRadar** — это приложение для сканирования локальной сети, которое обнаруживает сетевые устройства, собирает
информацию об их IP, MAC, DNS, производителе (по OUI) и проверяет доступность SNMP. Приложение имеет удобный графический
интерфейс, созданный с использованием JavaFX.

## Особенности

- **Постоянное сканирование.** В отличие от других сканеров, LANRadar осуществляет непрерывное сканирование сети, что
  позволяет моментально отображать изменения: новые подключения выделяются зелёным, отключённые устройства — красным, а
  изменения в данных — жёлтым.
- **Сканирование сети.** По заданному диапазону IP приложение обнаруживает активные устройства.
- **Определение производителя.** По MAC-адресу производится поиск информации об организации с использованием
  CSV-файла `ouiMAC.csv`.
- **Проверка SNMP.** Возможность проверки доступности SNMP-порта и выполнения SNMP Walk для получения подробной
  информации.
- **Графический интерфейс.** Удобное отображение результатов сканирования с возможностью копирования данных.

## Требования

- **Java 21** или выше
- **JavaFX SDK (скачать с openjfx.io)**
- **Maven** для сборки проекта

## Установка и настройка

1. **Клонируйте репозиторий:**

   ```bash
   git clone https://github.com/LeeRoyod/LANRadar.git
   cd LANRadar

2. **Настройте путь к JavaFX SDK:**
    - Откройте файл `pom.xml`.
    - Найдите свойство `javafx.sdk` и измените путь на актуальный для вашей системы.

      **Пример:**
      ```xml
      <properties>
        <javafx.sdk>C:/Program Files/Java/javafx-sdk-21.0.5</javafx.sdk>
      </properties>
      ```

## Сборка и запуск

1. **Сборка проекта:**  
   Выполните в терминале команду:
   ```bash
   mvn clean install

После успешной сборки `LANRadar.exe` будет доступен в папке `target/output`.

2. **Запуск приложения:**

- Перейдите в папку `target/output`.
- Запустите `LANRadar.exe`.

## Использование

1. **Запуск сканирования**
    - Выберите подсеть из списка доступных сетевых интерфейсов или введите свою (`ip/mask`, например, `192.168.0.99/24`).
    - Нажмите **Сканировать**.
    - Повторное нажатие остановит процесс сканирования.

2. **Просмотр результатов**
    - В таблице отображаются **IP**, **MAC**, **производитель**, **DNS-имя** и статус **SNMP**.
    - Если SNMP доступен, появится активная ссылка **Walk** — кликните, чтобы выполнить **SNMP Walk**.
